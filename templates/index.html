TopoIndex Web App

Implementasi Python dari Utilitas TopoIndex (USGS) untuk Analisis Stabilitas Lereng

Mulai Analisis Sekarang

Tentang Aplikasi Ini

Aplikasi web ini adalah implementasi Python dari utilitas TopoIndex, sebuah komponen pra-pemrosesan data topografi untuk model TRIGRS (Transient Rainfall Infiltration and Grid-Based Regional Slope-Stability) dari U.S. Geological Survey (USGS).

Indeksasi Topografi

Mengurutkan sel grid berdasarkan elevasi untuk memahami urutan aliran air.

Koreksi Arah Aliran

Memperbaiki dan memastikan data arah aliran konsisten secara hidrologis.

Perhitungan Faktor Bobot

Menghitung faktor pembobotan untuk mendistribusikan aliran permukaan (runoff).
